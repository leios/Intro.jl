Backspace # to go back to the regular REPL
julia> using Plots
[ Info: Precompiling Plots [91a5bcdd-55d7-5caf-9e0b-520d859cae80]

It took a while, but everything seems to be precompiled. Just for fun, let's close Julia and use Plots again.

julia> exit()
julia --project 
julia> using Plots

Hmm... That also took a while. Not as long as before, but it wasn't quick.

As it turns out, Julia's precompile time is sometimes quite long. If you are making a bunch of changes to a package, you might need to recompile your code again and again while bug hunting. That kinda sucks.
